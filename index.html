Getting started

Up and running in minutes.

No dashboard needed. Just invite and configure from Discord.

1
Create a bot on Discord Go to discord.com/developers/applications, create a new application, and add a bot. Enable all Privileged Intents.
2
Grab your token & invite the bot Copy your bot token from the Bot tab. Generate an invite URL with Administrator permissions and add the bot to your server.
3
Install dependencies & run Run pip install discord.py then python bot.py. Your bot is live.
4
Configure in Discord Use $setwelcome, $ticketsetup, $antinuke enable, and $autorole add to set up your server.
5
Host it 24/7 for free Deploy on Railway, Replit, or Oracle Cloud free tier so your bot stays online around the clock.
